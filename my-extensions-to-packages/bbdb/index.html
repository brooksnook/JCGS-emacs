bbdb-add-mailrc.el
Add the current entry to .mailrc
bbdb-export-lists.el
Create lists such as phone lists and Christmas card addresses
bbdb-families.el
Try to work out who is related to whom, using some conventions of mine
bbdb-grab-data.el
Grab things from email; there is probably stuff for this already in modern versions of bbdb
bbdb-gulag.el
Maintain a gulag of BBDB entries who have been unpersoned; BBDB will check for someone being in here, if they're not in your main DB. This way, you can keep the size of your main DB down. See also bbdb-stalin.el.
bbdb-human-names.el
Handle human names, using "graduate" and "ordained" fields to stick "Revd Dr" etc onto names, for use in bbdb-export-lists.el when it makes up address labels etc.
bbdb-iterators.el
Iterate over entries.
bbdb-reconstruct-lists.el
Undo some kind of damage to the daatbase, using an old copy that had been stored elsewhere.
bbdb-stalin.el
Exile entries to the gulag -- see also bbdb-gulag.el.
resort-bbdb.el
Looks like something else for undoing BBDB damage.
