Saving and restoring emacs state

save-emacs-state.el
Save the state of your Emacs session. this uses ../basics/load-directory.el to run a series of state saving modules, each of which should Create a file which restores one aspect of your emacs state. These files are loaded at start time, using load-directory.
savebuffer.el
Save the state of one Emacs buffer. This is used by save-emacs-state -- see above.
restorebuffer.el
Restore the state of one Emacs buffer. This is called from the file generated by save-buffer -- see above.
save-kill.el
Save the state of your kill-ring.

You can find the state saving modules I use, at ../savers1/*.el and ../savers2/*.el.
